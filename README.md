# 🔬 Scientific Calculator

A **Scientific Calculator** built using pure **HTML**, **CSS**, and **JavaScript**. It performs both basic arithmetic operations and advanced mathematical functions such as trigonometry, logarithms, exponentials, and more — all within a clean and responsive user interface.

This project is ideal for anyone learning front-end development or interested in how calculators work under the hood.

## 📌 Features

- ✅ Basic arithmetic operations (+, −, ×, ÷)
- ✅ Trigonometric functions (sin, cos, tan)
- ✅ Logarithmic and exponential functions
- ✅ Power and root calculations
- ✅ Bracket support for complex expressions
- ✅ Keyboard input support
- ✅ Responsive and user-friendly design
- ✅ Built with only HTML, CSS, and JavaScript (no libraries or frameworks)
